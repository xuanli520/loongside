## First Successful Outcome

The shortest supported path is still:

1. Install Loong.
2. Run guided onboarding.
3. Ask for one useful result.
4. Use `doctor` if setup is unhealthy.

The public docs lead with the base CLI path first. Channel setup, gateway surfaces, and richer runtime behavior come after the local assistant path is healthy.

## Reading Rule

- Install first.
- Run guided onboarding before touching deeper config.
- Use `doctor` as the repair path when the base CLI path does not feel healthy.
- Add providers, channels, and richer runtime surfaces only after the local assistant path works.
